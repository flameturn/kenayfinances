KenayFinanças
Descrição
KenayFinances é uma aplicação web simples e intuitiva para controle financeiro pessoal. Desenvolvida com HTML, CSS e JavaScript puro, permite adicionar, editar e remover transações, calcular saldos automaticamente (entradas, saídas e total) e salvar os dados localmente no navegador usando localStorage. Ideal para quem busca uma ferramenta leve e sem dependências externas para organizar finanças diárias.

O design é responsivo, moderno e inclui uma imagem de fundo personalizada para uma experiência visual agradável.

Funcionalidades
Adicionar transações: Cadastro de entradas ou saídas com descrição, valor, tipo e dados.
Editar transações: Modifique transações existentes diretamente na tabela.
Remove transações: Exclua transações indesejadas com um clique.
Cálculo automático de saldos: Visualize entradas, saídas e saldo total em tempo real.
Persistência de dados: Todas as transações são salvas no localStorage do navegador, garantindo que os dados persistam entre sessões.
Interface responsiva: Funciona bem em desktops, tablets e smartphones.
Validação de dados: Impede entradas inválidas, como valores negativos ou campos vazios.
Interação via modal: Formulário intuitivo para adicionar/editar transações.

Linguagem 
HTML5: Estrutura da página.
CSS3: Estilização, incluindo gradientes, sombras e responsividade.
JavaScript (ES6+): Lógica para manipulação de DOM, cálculos e armazenamento local.
localStorage: Para persistência de dados no navegador.
Como Usar
Baixe os arquivos:

Salve o código HTML como index.html.
Baixe a imagem de fundo ( 0_AU3rmB08Vntf33Is.jpg) e coloque na mesma pasta do index.html. (O link da imagem está embutido no código; certifique-se de que ela é acessível.)
Abra no navegador:

Abra o arquivo index.htmlem qualquer navegador moderno (Chrome, Firefox, Edge, etc.).
A página será carregada automaticamente, exibindo os saldos iniciais (zerados) e uma tabela vazia de transações.
uma transação:

Clique em "+ Nova transação".
Preencha os campos no modal (descrição, valor, tipo e dados).
Clique em "Salvar". A transação será apresentada na tabela e os saldos serão atualizados.
Edite uma transação:

Na tabela, clique no ícone ✏️ (editar) ao lado da transação desejada.
O modal abrirá preenchido com os dados atuais. Faça as alterações e salve.
Remoção de uma transação:

Clique no ícone ✕ (removedor) na tabela. A transação será mantida e os saldos recalculados.
Visualize os saldos:

Os cartões na parte superior mostram entradas, saídas e saldo total, atualizados automaticamente.
Pré-requisitos
Navegador web moderno com suporte a JavaScript e localStorage (todos os navegadores atuais suportados).
Não há necessidade de servidor ou instalação de software adicional.
